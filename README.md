# Netflix Movies & TV Shows – Exploratory Data Analysis

## Objective
This project performs Exploratory Data Analysis (EDA) on the Netflix Movies and
TV Shows dataset to identify trends, patterns, and insights using data visualization
and statistical analysis.

## Tools Used
- Python
- Jupyter Notebook
- Pandas
- Matplotlib
- Seaborn

## Key Analysis Performed
- Movies vs TV Shows distribution
- Content rating analysis
- Year-wise content addition
- Country-wise content production
- Movie duration analysis

## Conclusion
The analysis reveals Netflix’s focus on movies, mature-rated content, and rapid
growth in recent years. This project demonstrates effective data cleaning,
visualization, and interpretation techniques.
